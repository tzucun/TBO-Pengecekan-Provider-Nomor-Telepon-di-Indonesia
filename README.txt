Program ini dibuat untuk memenuhi Tugas Akhir Mata Kuliah Teori Bahasa dan Otomata (TBO)
Dibuat oleh: Kelompok 2

Cara menjalankan:
1. Pastikan python sudah terinstall dan versi terbaru
2. jalankan "pip install flask"
3. lalu jalankan app.py dengan mengetik "python app.py" di terminal
4. Klik localhost yang ada di terminal untuk menampilkan program ini
